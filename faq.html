  • How does the tool work??

    If the Intelligent Demographics device does not have any network connection, it starts a Bluetooth Server. The app discovers that Bluetooth Signal, and connects to it. Then, the device sends over all the WiFi networks it can currently connect to. The user chooses the WiFi he wants to connect to and enters the password for it. If the connection is successful, the user will see the IP address of the device and the Bluetooth server will stop broadcasting.

  • Why does the app ask for permission to access my location?

  • I can't find any INTELLIGENT DEMOGRAPHICS DEVICES

    • The booting process of the device takes some time, please be patient and try back in a couple minutes.
    • The Intelligent Demographics App requires access to your phone's Bluetooth to do it's magic. Your phone will ask about permissions to use the localization service. Make sure you authorize it.
    • The device is may already be connected to a LAN or WiFi.
    • You need to be near the device that you want to setup.

  • I CAN'T FIND THE WIFI NETWORK I WANT TO CONNECT TO

    • The WiFi signal may be too weak.
    • The WiFi technology might not be supported by the device. Currenty supported WiFi b/g/n (2,4GHz) with WPE, WPA, WPA2 encryption.

    • How can I use BerryLan on my already existing Raspbian image?

    You would need to install the BerryLan networkmanager and restart the Raspberry Pi.

    pi:~$sudo apt-get install dirmngr
pi:~$echo "deb http://repository.nymea.io stretch main" | sudo tee /etc/apt/sources.list.d/nymea.list
pi:~$sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-key A1A19ED6
pi:~$sudo apt-get update
pi:~$sudo apt-get install nymea-networkmanager
pi:~$sudo systemctl disable dhcpcd
pi:~$reboot

  • THE DEVICE DOESN'T CONNECT TO THE WIFI

    The WiFi signal may be too weak.

  • I GET A "PASSWORD IS WRONG" WHEN I TRY TO LOGIN

    Make sure you check your caps lock, or just press the "eye" next to the input field and check carefully if it is the correct password.
